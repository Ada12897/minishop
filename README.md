# Online Store Website

## Тема проекта
Мини-магазин

## Описание
Веб-сайт интернет-магазина с каталогом товаров, корзиной и простой системой аккаунта.
Проект реализован на чистом HTML, CSS и JavaScript без фреймворков.
Данные пользователя и корзины сохраняются с помощью localStorage.

## Страницы сайта
- index.html — главная страница
- catalog.html — каталог товаров
- product.html — страница отдельного товара
- cart.html — корзина
- account.html — аккаунт пользователя
- about.html — о магазине
- contacts.html — контакты

## JavaScript функционал
- загрузка и отображение товаров
- добавление и удаление товаров из корзины
- подсчёт общей стоимости корзины
- сохранение корзины в localStorage
- автоматическое сохранение данных пользователя (имя, адрес)
- отображение аккаунта пользователя на сайте
