# Online Store Vinyl

Небольшой фронтенд-проект интернет-магазина.

## О проекте
Сайт имитирует работу онлайн-магазина с каталогом товаров и корзиной.  

## Функционал
- каталог товаров
- добавление товаров в корзину
- подсчёт общей стоимости
- простая система аккаунта через localStorage
- сохранение данных пользователя между сессиями

## Технологии
- HTML
- CSS
- JavaScript (vanilla)
- localStorage
